Ejercicio 1: Calculadora Basica (DONE)
Descripcion: Crea una calculadora basica que permita a los usuarios sumar, restar, multiplicar y dividir dos numeros. La interfaz debe tener dos campos de entrada para los numeros y botones para cada operacion. Muestra el resultado en la pagina cuando se presione un boton.

Ejercicio 2: Conversor de Unidades (DONE)
Descripcion: Disena una pagina que permita convertir entre diferentes unidades de medida (por ejemplo, metros a pies, litros a galones, etc.). La pagina debe tener campos de entrada para el valor y la unidad de origen, asi como un boton para realizar la conversion y mostrar el resultado.

Ejercicio 3: Lista de Tareas (DONE)
Descripcion: Crea una lista de tareas donde los usuarios puedan agregar nuevas tareas, marcar tareas como completadas y eliminar tareas. Las tareas deben mostrarse en una lista, y las tareas completadas deben tener un estilo diferente (por ejemplo, tachado).

Ejercicio 4: Temporizador (Timer)
Descripcion: Disena un temporizador que permita a los usuarios establecer una cuenta regresiva en minutos y segundos. Cuando el temporizador llegue a cero, muestra una alerta. Incluye botones para iniciar, pausar y reiniciar el temporizador.

Ejercicio 5: Juego de Memoria
Descripcion: Crea un simple juego de memoria con tarjetas volteadas. El juego debe contener un numero par de tarjetas boca abajo, con cada tarjeta teniendo una pareja. Los usuarios pueden hacer clic en dos tarjetas a la vez para voltearlas, y si las tarjetas coinciden, se quedan volteadas; de lo contrario, se vuelven a voltear.

Ejercicio 6: Galeria de Imagenes
Descripcion: Disena una galeria de imagenes que muestre una serie de miniaturas. Cuando se hace clic en una miniatura, muestra la imagen en tamano completo en una seccion destacada de la pagina. Agrega botones para navegar a la imagen anterior y siguiente.

Ejercicio 7: Formulario de Contacto con Validacion
Descripcion: Crea un formulario de contacto que incluya campos para el nombre, el correo electronico y el mensaje. Implementa validacion en el cliente para asegurar que los campos no esten vacios y que el correo electronico tenga un formato valido. Muestra mensajes de error adecuados si la validacion falla.

Ejercicio 8: Tablas y Graficos Dinamicos
Descripcion: Crea una pagina que permita a los usuarios ingresar datos en una tabla y luego visualizar esos datos en un grafico (puedes usar una biblioteca como Chart.js). La pagina debe tener campos de entrada para agregar nuevas filas a la tabla y un boton para actualizar el grafico con los datos de la tabla.

Ejercicio 9: Pagina de Perfil Dinamica
Descripcion: Disena una pagina de perfil que muestre informacion de un usuario (nombre, foto, biografia, etc.). Incluye un formulario que permita a los usuarios editar esta informacion y actualizar la pagina en tiempo real con los nuevos datos ingresados.

Ejercicio 10: Reproductor de Audio
Descripcion: Crea un reproductor de audio que permita a los usuarios reproducir, pausar y ajustar el volumen de una pista de audio. Incluye una barra de progreso que muestre el tiempo transcurrido y el tiempo total de la pista.
